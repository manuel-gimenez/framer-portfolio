# Skills

Reusable skills I built while making this Framer portfolio. They're kept here —
rather than buried in my global tooling — because each one came out of a real
problem I hit on this project, and together they show how I work, not just what
I shipped.

Each skill folder contains:

- a `SKILL.md` (the machine-readable instructions an AI assistant follows), and
- a short write-up of **the problem it solves** and **why**.

## Skills

| Skill | Problem it solves |
| --- | --- |
| [`escape-hatch/`](./escape-hatch) | Getting unstuck fast: when 3+ same-level fixes fail, step back and re-solve the problem at a different level (inside / around / replace) instead of retrying harder. The worked example is the Lottie letterbox problem from this very repo. |
